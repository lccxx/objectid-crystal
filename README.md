# objectid-crystal

simple implement mongodb bson objectid

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  objectid-crystal:
    github: liu-chong/objectid-crystal
```

## Usage

```crystal
require "objectid"

ObjectId.new
```
